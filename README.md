# Islam pro

Application islamique mobile-first avec:

- Horloge + date en direct.
- Horaires de prière + minutes avant la prochaine prière.
- Podcasts, histoires pour dormir, histoires du Prophète ﷺ.
- Hadith du jour et verset du jour.
- Calendrier des fêtes/événements.
- Lien de partage sécurisé généré avec suffixe aléatoire 64 bits.

## Démarrage

```bash
python3 -m http.server 4173
```

Puis ouvre http://localhost:4173
